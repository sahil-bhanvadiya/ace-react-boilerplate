## How to setup 

Clone this repo.

Install the dependencies:

```bash
pnpm install
```

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
